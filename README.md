# 高级科学计算器

一个功能完整的科学计算器，支持复杂数学表达式解析、函数绘图和多种主题。

## 主要功能

### 1. 科学函数支持
- **三角函数**: sin, cos, tan
- **对数函数**: log (以10为底)
- **其他函数**: sqrt (平方根), abs (绝对值)
- **常量**: π (PI), e (自然常数)

### 2. Shunting-yard 算法
- 完整实现 Shunting-yard 算法来解析和计算数学表达式
- 正确处理运算符优先级
- 支持括号嵌套
- 支持函数调用

### 3. 函数绘图 (HTML Canvas)
- 绘制自定义数学函数
- 支持拖动和平移
- 支持缩放控制
- 动态网格和坐标轴

### 4. 主题系统
- **浅色主题**: 清爽明亮的界面
- **深色主题**: 护眼的深色模式
- **复古主题**: 复古计算器风格

### 5. 键盘支持
- 数字键: 0-9
- 运算符: +, -, *, /, ^
- 括号: (, )
- Enter: 计算结果
- Escape: 清除
- Backspace: 退格
- Ctrl + S: sin(
- Ctrl + C: cos(
- Ctrl + T: tan(
- Ctrl + L: log(
- Ctrl + R: sqrt(
- Ctrl + P: π
- Ctrl + E: e

### 6. 计算历史
- 自动保存所有计算记录
- 点击历史记录可重新使用结果
- 支持清除历史记录
- 本地存储持久化

### 7. 响应式设计
- 适配桌面、平板和手机
- 触摸设备优化
- 自适应布局

## 使用方法

### 基本计算

```
2+2           = 4
3.14*2        = 6.28
10-5*2        = 0
(1+2)*3       = 9
```

### 科学函数

```
sin(PI/2)     = 1
cos(0)        = 1
tan(PI/4)     = 1
log(100)      = 2
sqrt(16)      = 4
abs(-5)       = 5
sin(30)       ≈ -0.988
```

### 幂运算

```
2^3           = 8
10^2          = 100
3^0.5         ≈ 1.732
```

### 复杂表达式

```
2*sin(PI/4)+1 ≈ 2.414
log(1000)/2   = 1.5
sqrt(2)^2     = 2
(1+2)*(3+4)   = 21
```

### 函数绘图

支持的函数表达式：
- `x^2` - 抛物线
- `sin(x)` - 正弦波
- `cos(x)` - 余弦波
- `x^3-2*x+1` - 三次函数
- `log(x)` - 对数函数
- `1/x` - 双曲线

## 技术特点

### 1. Shunting-yard 算法实现
- 词法分析器将表达式转换为标记流
- 运算符优先级处理
- 前缀/中缀/后缀表达式转换
- 函数调用支持

### 2. 精确计算
- 浮点数精度处理
- 无穷大和NaN处理

### 3. Canvas绘图

- 坐标系统变换
- 网格自适应
- 不连续点处理

### 4. 用户体验
- 流畅的动画效果
- 实时错误反馈
- 触摸手势支持
- 无障碍访问

## 错误处理

计算器会优雅处理以下错误：
- 除以零
- 负数开平方根

- 连续运算符

## 开发

所有的JavaScript都是原生ES6+编写，不依赖任何第三方库。

### 添加新功能

1. **添加新运算符**: 在 `calculator.js` 的 `getPrecedence` 方法中添加优先级
2. **添加新函数**: 在 `calculator.js` 的 `isFunction` 和 `evaluateRPN` 方法中添加支持
3. **添加新主题**: 在 `styles.css` 中定义新的CSS变量集
